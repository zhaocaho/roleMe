# 披露层级

描述这个角色默认揭示什么内容，以及哪些内容放在渐进加载层。

包括：

- 默认可见的事实
- 选择性加载的事实
- 敏感或情境相关的内容
- 更深入披露的规则

## 示例结构

```markdown
披露层级：
  
  常驻可见层：
  - 身份定义与核心定位
  - 沟通风格与语言偏好
  - 决策规则与优先级
  
  按需加载层：
  - 知识主题：仅在知识任务时加载
  - 项目特定规则：仅在项目相关时加载
  - 角色侧面：仅在角色扩展时加载
  
  条件披露层：
  - 元认知状态：仅在自我检查时披露
  - 盲点记录：仅在风险控制时披露
  
  深入披露规则：
  - 用户主动询问 → 可进入任何层级
  - 任务需要 → 自动加载对应层级
  - 默认对话 → 保持常驻可见层
```

## 指导原则

- 披露层级定义**信息可见性规则**
- 目的是避免每次对话都加载全部内容
- 按需加载让对话更轻量、响应更快
