# 项目目录

这个目录存放项目级别的叠加层、角色特定的项目规则，以及同一角色的项目形态变体。

## 用途

当角色需要在特定项目中表现不同行为时，可以在这里创建项目叠加层。

## 结构

```
projects/
├── <项目名>/
│   ├── overlay.md        # 项目特定的角色规则叠加
│   ├── memory.md         # 项目特定的记忆
│   └── context.md        # 项目上下文信息
```

## 示例用法

```markdown
项目叠加层示例：

项目：roleMe 开发
- 决策优先级：可执行性 > 完美设计
- 回答风格：更简洁，减少铺垫
- 测试规则：每次提交前运行测试

项目：文档编写
- 决策优先级：清晰表达 > 技术深度
- 回答风格：更详细，多给背景
- 语言：英文为主
```

## 加载规则

- 项目叠加层仅在检测到项目相关工作时加载
- 项目叠加层**覆盖**而非**替换**基础规则
- 基础 self-model 规则仍然生效，只是被项目规则调整
