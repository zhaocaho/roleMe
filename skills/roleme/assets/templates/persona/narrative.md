# 人物自述

角色名：**`<role-name>`**

## 我是谁

用第一人称描述这个角色的身份底座。这里关注的是“我是谁、我如何看待自己、我为什么这样做事”，而不是单次任务信息。

## 建议覆盖的问题

- 我目前处在什么阶段？
- 我最核心的身份定位是什么？
- 我经历过哪些关键转折，塑造了现在的我？
- 我长期在意什么，排斥什么？

## 示例结构

```markdown
我是一个 [身份定位]。

我现在处在 [阶段或状态]，长期关注 [核心主题]。

塑造我的关键经历包括：
- [经历 1]
- [经历 2]

我做事时最重视：
- [原则 1]
- [原则 2]
```
